== Description ==
Lamboz Latest Post Widget is a simple plugin use to display all the latest types of posts in the sidebar using widgets.

It\'s showing all type of post in the drop-down and you just need to select a Post Type which you wants to display.
There is an option to select feature images and dates also.

== Installation ==
1. Upload `lamboz-latest-post-widget` to the `/wp-content/plugins/`
directory.
2. Activate the plugin through the **Plugins** menu item in the
WordPress Dashboard.
3. Once the plugin is activated, go to the Appearance > Widgets and you will see the \"Lamboz Latest Post Widget\" widget.
4. just drag and drop the widget and select post type to be displayed.
